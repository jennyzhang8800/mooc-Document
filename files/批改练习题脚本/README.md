罗富文的选择题批改脚本。

【使用方法】

1. 先clone teacher/answer.git到本地
2. 进入到tool目录
3. 修改conf.py中的repopath为answer仓库的实际地址
4. 运行grade.py批改选择题
5. git push 同步仓库
